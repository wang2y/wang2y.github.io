Reggie外卖项目

一、概述

技术层面

  • 本次写的reggie项目使用的技术有Spring、SpringBoot、SpringCache、Vue、Mysql、Redis、ElementIU、Mybatis-plus、git(代码托管在gitee仓库，国内较快)、Lombok、Druid、aliyun短信验证服务(付费)、aliyun服务器

工具层面

  • 使用了Maven作为依赖管理的工具，idea作为代码开发工具

版本

  • jdk8、Springboot2.x、mysql8、idea2022、maven3.8.6

二、数据库设计

1.表设计

1.概述

​ 为完成开发需求设计了11张表，分别为employee(后台管理系统登录，管理员及员工)、dish、dish_flavor、category、setmeal、setmeal_dish、user(用户)、order、order_detail、address_book、shopping_cart(购物车)。

2.执行sql脚本

​ 因为表设计不是开发重点，直接在navigate中执行sql脚本，生成表文件，具体字段和约束见文件。

3.表说明

(1)employee

id name username password phone sex id_number status create_time update_time create_user update_user
主键值 员工姓名 账号 密码 手机 性别 身份证 状态 创建时间 更新时间 创建用户 更新用户

注：sex取值0为女、1为男

​ status取值0为禁用、1为启用

(2)dish

id name category_id price code image description status sort create_time create_user update_time update_user is_delete
主键值 菜品名称 菜品分类主键值 价格 编码 图片地址 描述 状态 排序规则 创建时间 创建用户 更新时间 更新用户 是否删除

注：status取值0为禁用、1为启用

​ is_delete取值0为未删除、1为已删除

(3)dish_flavor

id dish_id name value create_time update_time create_user update_user is_delete
主键值 菜品主键值 口味名称 口味值 创建时间 更新时间 创建用户 更新用户 是否删除

(4)category

id type name sort create_time update_time create_user update_user
主键值 类型 名称 排序规则 创建时间 更新时间 创建用户 更新用户

注：type取值1为分类、2为套餐
